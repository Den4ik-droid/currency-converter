# Конвертер валют (BE)

Бэккенд-приложение для помощи в конвертации валют. Является прослойкой между
клиентским приложением и сторонними API, предоставляющими инфо о курсах валют.

## Реализовано на данный момент

- Базовая структура backend‑приложения (Node.js + Express + TypeScript)
- Маршруты:
  - `GET /api/user`
  - `POST /api/user`
- Middleware для работы с cookie (`user_id`)
- Временное хранилище пользователей (Map)
- Простая валидация входящих данных
- Docker Compose для быстрого запуска проекта
- Добавлены файлы конфигурации: `.env.example`, `.nvmrc`, линтер
- Подготовлена демонстрация работы эндпоинтов в папке `demo`


## Планируется реализовать далее

- Подключение постоянного хранилища данных (Supabase / Firebase)
- Маршрут `GET /api/currencies` — получение списка валют с кешированием на 1 час
- Маршрут `GET /api/rates` — получение курсов валют с поддержкой параметров base и targets
- Реализация логики выбора базовой валюты (из настроек пользователя или USD по умолчанию)
- Кэширование курсов валют:
  - 24 часа (хранение в БД)
  - 5 минут для повторных запросов одного пользователя (хранение в памяти)
- Swagger‑документация API
