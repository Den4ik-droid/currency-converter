# Конвертер валют (BE)

Бэккенд-приложение для помощи в конвертации валют. Является прослойкой между
клиентским приложением и сторонними API, предоставляющими инфо о курсах валют.

## Реализовано на данный момент

- Базовая структура backend-приложения (Node.js + Express + TypeScript) 
- Маршруты(фото демонстрации в demo/demo.docx): 
- `GET /api/user` — получение настроек пользователя 
- `POST /api/user` — обновление настроек пользователя 
- Middleware для работы с cookie (`user_id`) 
- Временное хранилище пользователей (Map) 
- Валидация входящих данных 
- Docker Compose для быстрого запуска проекта 
- Корректный .gitignore
